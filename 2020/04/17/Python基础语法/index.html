    • 倒序：sorted(ls,resever=True)

特殊的形参

  1. 默认值：def method(a, b=’5’):
  • 在调用时，可以不给形参b赋值，method(5)；– 默认参数
  • 也可以method(5,6)来覆盖默认值 – 位置调用
  • 也可以method(b=6, a=7)调用 – 关键字调用
  1. 无限制形参

2-1 def method(a, *b)

  • *b被认为是一个元组
  • 当调用method(5, 5,5,5)时，被认为 method(5, (5,5,5))

2.2 def method(a, **b)

  • *b被认为是一个字典
  • 当调用method(5, a=5,b=5,c=5)时，被认为 method(5, {‘a’=5,’b’=5,’c’=5})

形参与实参的对应联系

位置对应

让实参按照形参的位置一一对应传进去

关键字对应

调用时，声明形参名=实参，例如method(b=6, a=7)调用

函数修改列表以及禁止函数修改列表

当调用method(ls,ls1)时，在函数内部修改ls，即对ls列表本身做了操作；

如果不想对原列表做出操作，需要把列表的副本传入参数

method(ls[:], ls1[:])

变量与对象

传值和传引用是函数参数传递的两种方式，在Python中参数是如何传递的？

Python 中一切皆为对象，数字是对象，列表是对象，函数也是对象，任何东西都是对象。而变量是对象的一个引用（又称为名字或者标签），对象的操作都是通过引用来完成的。例如，[]是一个空列表对象，变量 a 是该对象的一个引用

在 Python 中，「变量」更准确叫法是「名字」，赋值操作 = 就是把一个名字绑定到一个对象上。就像给对象添加一个标签。

函数编写规范

  1. 给函数描述性的名称，且只在其中使用小写字母和下划线
  2. 定义时，给定默认值时，等号两侧不要有空格
  3. 传递实参时，关键字实参调用等号两侧不要有空格

类

类的编写

  1. Python中，首字母的写的名称代表类；class Dog():，定义中括号是空的（除非他继承于某个类）
  2. 每当根据类创建示例时都会自动运行__init__()方法，相当于构造函数，Python的类只有一个构造函数，函数中必须有self形参，它代表着这个类当前的实例。
  3. 类中的每个属性都必须要有初始值，无论是在__int__()方法里赋值的还是在形参中设置默认值的。

类的继承

一个类继承另外一个类时，它将自动获得另一个类的所有属性和方法

  1. 子类的__init__()方法需要父类施以援手；super().__init__(xxx)
  2. 创建子类时，父类必须在此文件中，并且位于子类前面
  3. 定义子类时，必须在括号里指定父类的名称；class ElectrucCar(Car):
  4. 子类可以重写父类的方法；Ctrl + o

文件

如果要以任何方式使用文件：

  1. 首先要打开文件 with open('xxx.txt') as file:，使用open()打开文件，默认以只读方式打开
  2. 关键字with在不需要访问文件资源后将其关闭
  3. 然后在使用read()方法来读取文件的内容

读取文本文件时，python将其中的所有文本都解读为字符串。如果你读取的是数字，并要将其作为数值使用，就必须用int()/float()函数转换。

函数 返回 描述

readlines() [] 返回文件各行内容构成的列表

json模块的函数

函数 描述

dumps(x) 将字典类型{}/[{},{}]的变为json
load(x) 将json变为对象（列表或字典）

Encode过程，是把python对象转换成json对象的一个过程，常用的两个函数是dumps和dump函数。两个函数的唯一区别就是dump把python对象转换成json对象生成一个fp的文件流，而dumps则是生成了一个所有程序语言都认识的字符串。
Decode过程，是把json对象转换成python对象的一个过程，常用的两个函数是loads和load函数。区别跟dump和dumps是一样的。
