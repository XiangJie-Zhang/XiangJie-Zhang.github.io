Shell概述

Shell解析器

Linux提供的Shell解析器有：

  1. /bin/sh
  2. /bin/bash
  3. /bin/rbash
  4. /bin/dash
  5. /usr/bin/tmux
  6. /usr/bin/screen

Ubuntu默认的解析器echo $SHELL是bash

Shell工具之cut、sed、awk、sort

cut

cut的工作就是“剪”，具体的说就是在文件中负责剪切数据用的。cut 命令从文件的每一行剪切字节、字符和字段并将这些字节、字符和字段输出。

基本用法

cut [选项参数] filename

说明：默认分隔符是制表符

选项参数说明

选项参数 功能
-f 列号，提取第几列
-d 分隔符，按照指定分隔符分割一行为多个分割列
-c 获取指定位置的字符(多个时n-n+k)

sed

sed是一种流编辑器，它一次处理一行内容。处理时，把当前处理的行存储在临时缓冲区中，称为“模式空间”，接着用sed命令处理缓冲区中的内容，处理完成后，把缓冲区的内容送往屏幕。接着处理下一行，这样不断重复，直到文件末尾。文件内容并没有改变，除非你使用重定向存储输出。

基础用法

sed [选项参数] ‘command’ filename

选项参数说明

选项参数 功能
-e 直接在指令列模式上进行sed的动作编辑(可多个指定)
-i 直接编辑文件

命令功能描述

command包含范围和操作

命令 功能描述
a 新增，新增的字符串在下一行展示
d 删除
s 查找、替换(s/代替换/替换为/g)

范围可以是行号(n|n,n+k)或者正则表达式(要用//包裹)。

🔺awk

一个强大的文本分析工具，把文件逐行的读入，以空格为默认分隔符将每行切片，切开的部分再进行分析处理。

基本用法

awk [选项参数] ‘pattern1{action1} pattern2{action2}...’ filename

pattern：表示AWK在数据中查找的内容，就是匹配模式

pattern可以是正则表达式(用//包裹)，还可以是BEGIN、END之类。

action：在找到匹配内容时所执行的一系列命令(用{}包裹)

选项参数

选项参数 功能
-F 指定每行分隔符
-v 复制用户自定义变量

内置变量

有很多，只列举一些常用！

变量 说明
FILENAME 文件名
NR 已读行数
NF 浏览记录的域的个数(切割后，列的个数)

sort

sort命令是在Linux里非常有用，它将文件进行排序，并将排序结果标准输出。

基本语法

sort(选项)(参数)

选项

选项 说明
-n 依照数值的大小排序
-r 以相反的顺序来排序
-t 设置排序时所用的分隔字符
-k 指定需要排序的列

ps：如果不指定-n默认以字符串字典排序

参数：指定待排序的文件列表
